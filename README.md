# Figma-to-Webpage-UI
A responsive webpage built from a Figma design using HTML and CSS. This project focuses on accurate UI implementation, clean code structure, modern styling, and responsive layouts. It demonstrates frontend development skills by transforming a design mockup into a functional and visually appealing web page.

## Features

* Pixel-perfect Figma design implementation
* Responsive layout for different screen sizes
* Clean and semantic HTML structure
* Modern CSS styling
* User-friendly interface

## Technologies Used

* HTML5
* CSS3

## Project Structure

```
project-folder/
│
├── index.html
├── style.css
└── assets/
```

## Learning Outcomes

* Converting UI designs into code
* Building responsive web layouts
* Writing clean and maintainable HTML/CSS
* Improving frontend development skills
